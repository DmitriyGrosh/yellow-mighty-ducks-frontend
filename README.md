# yellow-mighty-ducks-frontend

### Installation

Установить зависимости в проекте:
```sh
$ pnpm install
```

### Development

Запустить проект в режиме разработки ([localhost:443](https://localhost:443)):
```sh
$ pnpm dev
```

### Building

Сборка проекта в production-режиме:
```sh
$ pnpm prod
```
